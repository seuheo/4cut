## 📝 **테스트 체크리스트**

### **시나리오 1: 앱 실행 및 기본 네비게이션**
- [ ] 앱이 에뮬레이터에서 정상적으로 실행됨
- [ ] 홈 화면이 정상적으로 표시됨
- [ ] 하단 네비게이션 바가 정상적으로 작동함
- [ ] 각 탭(Home, Frame, Calendar, Settings, Profile) 간 전환이 정상적으로 작동함

### **시나리오 2: 사진 선택 기능**
- [ ] Frame 탭 진입 시 사진 선택 화면이 표시됨
- [ ] 갤러리 접근 권한이 정상적으로 요청됨
- [ ] 갤러리에서 사진 선택이 정상적으로 작동함
- [ ] 선택한 사진이 4컷 그리드에 정상적으로 표시됨
- [ ] 랜덤 사진 추가 기능(있는 경우)이 정상적으로 작동함
- [ ] 사진 교체/삭제 기능이 정상적으로 작동함

### **시나리오 3: 프레임 선택 기능**
- [ ] "다음 단계" 버튼 클릭 시 프레임 선택 화면으로 이동
- [ ] 프레임 목록이 정상적으로 표시됨
- [ ] 프레임 선택 시 미리보기가 정상적으로 표시됨
- [ ] Long Form 프레임이 전체적으로 보임 (잘리지 않음)
- [ ] 프레임 선택 후 결과 화면으로 자동 이동

### **시나리오 4: 결과 확인 및 저장**
- [ ] 합성된 이미지가 정상적으로 표시됨
- [ ] "저장하기" 버튼이 정상적으로 작동함
- [ ] 갤러리에 이미지가 정상적으로 저장됨
- [ ] 성공 메시지가 표시됨
- [ ] 앱이 크래시 없이 정상적으로 작동함
