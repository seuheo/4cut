## 💡 **사용자 테스트 가이드**

### **에뮬레이터에서 테스트하려면**:

1. **Android Studio에서 에뮬레이터 실행**
   - Medium_Phone_API_36.0 (AVD) - 16

2. **앱 실행**
   - 앱 아이콘을 눌러 실행
   - 홈 화면이 정상적으로 표시되는지 확인

3. **Frame 탭으로 이동**
   - 하단 네비게이션 바에서 "Frame" 탭 선택
   - 사진 선택 화면 진입

4. **사진 4장 선택**
   - 갤러리에서 사진 선택 또는 "랜덤 사진 추가" 버튼 클릭
   - 사진이 4컷 그리드에 표시되는지 확인

5. **"다음 단계" 버튼 클릭**
   - 프레임 선택 화면으로 이동 확인

6. **프레임 선택**
   - 포맷 선택 (일반 / Long Form)
   - 원하는 프레임 선택
   - 결과 화면으로 자동 이동 확인

7. **결과 확인 및 저장**
   - 합성된 이미지 확인
   - "저장하기" 버튼 클릭
   - 성공 메시지 확인
