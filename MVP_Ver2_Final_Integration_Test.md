# MVP Ver2 최종 통합 테스트 계획

## 테스트 목표
- 앱의 전반적인 핵심 기능 유기적 연동 확인
- 수정된 버그들이 여전히 해결된 상태로 유지되는지 확인
- 신규 기능들이 기존 기능과 충돌하지 않는지 확인

## Phase 1: Happy Path - 전체 핵심 흐름 테스트

### Test Case 1.1: 사진 선택 → 프레임 선택 → 저장 (Happy Path)

**테스트 단계**:
1. 앱 실행
2. 프레임 탭 접근
3. 사진 4장 선택
4. 프레임 선택 화면 진입
5. 프레임 선택 ("인생 네컷" 또는 "Long Form")
6. 결과 화면에서 합성 이미지 확인
